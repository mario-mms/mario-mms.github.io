Proyecto de HTML y CSS

Para la creación de mi página Web o portfolio, he seguido los siguiente pasos:

  1. Boceto inicial sobre el diseño del menú y el contenido de las páginas.
  2. Estructuración del sistema de carpetas a partir de la raíz / .
  3. Creación del menú de navegación con su HTML y CSS correspondiente, así como todos los enlaces con rutas absolutas.
  4. Copiar el HTML y el CSS para todas las páginas, también debemos incluir los media queries.
  5. Diseño de la portada con las correspondientes etiquetas y CSS personalizado.
  6. Creación de las páginas de los diferentes módulos, así como las subpáginas de proyectos y prácticas. Con su propio CSS.
  7. Crear una página para cada proyecto o práctica.
  8. Revisar todas las rutas de los enlaces, para que la WEB sea perfectamente navegable.
    9. *Mediante Git y GitHub debemos hacer commit & push según vamos avanzando el trabajo, para evitar la pérdida de información por error o accidente.*
  9. Una vez subida a GitHub realizaremos la publicación en GitHub Pages.
